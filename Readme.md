# Predictive Modeling Using Machine Learning

## Project Overview
This project predicts student placement status using supervised machine learning techniques.

## Features
- Data preprocessing
- Missing value handling
- Duplicate removal
- Outlier detection using IQR
- Feature engineering
- Model training and testing
- Accuracy evaluation
- Confusion matrix visualization
- ROC curve analysis
- Feature importance analysis

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm
- Gradient Boosting Classifier

## Evaluation Metrics
- Accuracy Score
- Classification Report
- Confusion Matrix
- ROC Curve

## Expected Outcome
Gain experience in supervised learning and model evaluation.
